项目介绍

这是一个使用 Next.js 14 构建的现代化 Web 应用程序演示项目。 我们采用了最新的 App Router 架构，集成了 Tailwind CSS 进行样式设计， 并使用了 TypeScript 来提供更好的开发体验。

技术栈

  • Next.js 14 - React 全栈框架
  • React 18 - 用户界面库
  • TypeScript - 类型安全的 JavaScript
  • Tailwind CSS - 实用优先的 CSS 框架
  • App Router - 新的路由系统
